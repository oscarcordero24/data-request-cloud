To Get Gage Data:

  • Choose a basin from the dropdown menu.
  • Select a gage you want data for.
  • Select a data type (Daily or Hourly).
  • Choose a time window for the data.
  • Press "Get Gage Data" to extract the data.
  • Once the data is ready, press "Get Gage Excel" to generate and download an Excel file containing the gage's data.

To Get Basin Data:

  • Choose a basin from the dropdown menu.
  • Select "All Gages" from the "Gage" dropdown menu.
  • Select a data type (Daily or Hourly).
  • Choose a time window for the data.
  • Press "Get Basin Data" to extract the data.
  • Once the data is ready, press "Get Basin Excel" to generate and download an Excel file, with each tab containing data for a gage in the basin.
